# 🏏 IPL Data Analysis (2008 - 2022)

## 📌 Project Overview

This project performs an in-depth Exploratory Data Analysis (EDA) on the Indian Premier League (IPL) dataset covering seasons from 2008 to 2022. The objective is to uncover meaningful insights related to team performances, player achievements, toss decisions, and match outcomes using Python-based data analysis and visualization techniques.

The analysis helps identify patterns and trends that can support strategic decision-making in cricket analytics.

---

## 🎯 Objectives

- Analyze IPL match data from 2008 to 2022.
- Understand team performance across seasons.
- Identify top-performing batsmen and bowlers.
- Examine the impact of toss decisions on match outcomes.
- Discover trends and patterns using visualizations.
- Generate actionable insights through data-driven analysis.

---

## 📂 Dataset Information

This project uses two datasets:

### 1. IPL Matches Dataset

Contains match-level information such as:

- Match ID
- Season
- Venue
- Teams
- Toss Winner
- Toss Decision
- Match Winner
- Player of the Match
- Margin of Victory

### 2. IPL Ball-by-Ball Dataset

Contains ball-by-ball information including:

- Batter
- Bowler
- Runs Scored
- Extras
- Wickets
- Innings Details

### Dataset Source

https://www.kaggle.com/datasets/vora1011/ipl-2008-to-2021-all-match-dataset

## 📈 Key Insights

- Mumbai Indians have emerged as one of the most successful franchises in IPL history.
- Virat Kohli is among the leading run scorers in IPL.
- Dwayne Bravo is one of the highest wicket-taking bowlers.
- Most captains prefer to field first after winning the toss.
- Winning the toss often provides a competitive advantage.
- Certain venues significantly influence match outcomes.
